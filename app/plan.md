# 🗺️ 세계사 전황판 프로젝트 (World War Board) 설계도

## 1. 프로젝트 개요
* **목표:** 텍스트 중심의 지루한 역사가 아닌, 지도와 데이터가 실시간으로 연동되는 인터랙티브 역사 시뮬레이터 개발.
* **타겟:** 역사 애호가, 교육용 시각 자료가 필요한 학생 및 교사.
* **주요 테마:** (예: 제1차 세계 대전, 제2차 세계 대전, 한국전쟁 등 메인 테마 작성)

## 2. 핵심 기능 (Core Features)
1. **인터랙티브 역사 지도 (Mapbox 기반)**
   - 연도별 초정밀 국경선(GeoJSON) 렌더링.
   - 사건 클릭 시 해당 지역으로 부드럽게 카메라 이동 (FlyTo).
   - 세력권(동맹국/협상국 등) 색상 실시간 전환.
2. **다이내믹 타임라인 (Timeline)**
   - 연도별/사건별 카드 UI 구성.
   - 현재 선택된 사건 시각적 하이라이트.
3. **실시간 전력 분석 대시보드 (Stats Dashboard)**
   - 선택된 전투의 참전국, 투입 병력, 사상자 통계 실시간 표기.
4. **언어 및 UI 최적화**
   - 지도 내 지명 100% 한국어 패치.
   - 가독성 높은 미니멀리즘 디자인 적용 (라이트/다크 모드 등).

## 3. 데이터 구조 설계 (Data Architecture)
* **지도 데이터:** 깃허브 오픈소스 역사 GeoJSON URL 연동 (용량 최적화).
* **역사 데이터 (JSON 형태 분리 필요):**
  - `events`: 날짜, 제목, 좌표, 설명, 영향력.
  - `stats`: 참전국 이름, 투입 병력, 전사, 부상 데이터.

## 4. 개발 로드맵 (Milestones)
* **[Phase 1] 기반 공사 (현재 완료 단계):** 지도 띄우기, 국경선 그리기, 기본 타임라인 연동.
* **[Phase 2] 데이터 확장 및 UI 고도화:** (진행 예정) 통계 대시보드 개선, 컴포넌트 분리, 추가 연도 데이터 확보.
* **[Phase 3] 추가 디테일 구현:** (진행 예정) 사운드 효과, 모바일 반응형 완벽 대응, 툴팁 디테일 강화 등.
* **[Phase 4] 최종 배포:** 깃허브 및 Vercel을 통한 웹 호스팅.
